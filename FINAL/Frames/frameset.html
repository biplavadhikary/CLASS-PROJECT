FRAMESET

Frameset element is the collection of more than one frame element or a collection of more than one HTML file in a window browser.It has two attributes which helps in arrangement or organizing various frames in a window browser.


Framset Example
Attributes of Frameset
Attribute Value Description
cols -> pixels
-> %
-> *		 Specifies the number and size of columns in a frameset.
rows -> pixels
-> %
-> *		 Specifies a page that contains a long description of the content of a frame.
