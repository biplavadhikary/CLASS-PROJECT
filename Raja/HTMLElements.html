HTML Elements

  • An HTML element usually consists of a start tag and end tag, with the content inserted in between:

    • <tagname>Content... </tagname>

  • If the element contains other content, it ends with a closing tag.

  • The element name is preceded by a forward slash.As shown in below table:


Sart Tag Content End Tag
<p> This is paragraph content. </p>
<h1> This is heading content. </h1>
<div> This is division content. </div>

Void Element


  • There are some HTML elements which don't need to be closed,those type of elements are called void elements.

Example:

<img>.../>, <hr> /> and <br> />
