Table Encyclopedia

Introduction

A table is a 2 dimensional matrix containing rows and columns. Tables are intended for displaying data in coloumn on a web page. All table related tags are included between the table.../table tags. Each row of a table is decribed b/w tr.../tr tags. The data displayed inside the tr tags is kept inside the td.../td tags.

Table Rows can be of two types:

  1. Header rows: these are the heading given to the table and they span accross all the coloumn of the table. They are mentioned inside the th.../th Tags.
  2. Data rows: individual data cells placed int he horizontal plane creates a data row. There could be A single data cell or multiple data cells.

Data cells hold the data that must be displayed int the table. A data row is defined using tr.../tr tags. Text matter in a data cell is by default left aligned. Any special formating like boldface or italics is done by including appropriate formating tags inside the Tr.../tr tags. An image can also be displayed in a data cell.

The attributes included within the table tags are:

align Horizontal alignment of the row is handled by the align tag. It can align the content to either Left, right, center, justify or inherit places.
valign Vertical alignment of the row is handled by this tag. It can algin the content to either top,bottom Or middle places
width Sets the width to according to the given no. Of pixels or percentage. If nothing is mentioned Then it configures the size according to the text
border Controls the border of the row either to pe present or not and also its width in pixels
cellpadding Controls the distance b/w the cell walls and the text inside
cellspacing Controls the cell spacing from one another
colspan Controls the coloumn span of any data cell. Mentioned in the no. Of the coloumn to be covered
rowspan Controls the row span of any data cell. Mentioned in no. Of rows to be covered
